# 🎓 School Management System

![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)
![PyQt5](https://img.shields.io/badge/PyQt5-GUI-green.svg)
![SQLite](https://img.shields.io/badge/Database-SQLite-orange.svg)

A desktop **School Management System** built using **Python, PyQt5, and SQLite**.  
It allows managing students, marks, fees, and attendance in a simple graphical interface.

---

## 📌 Features

### 👨‍🎓 Student Management
- Add new students
- Edit student information
- Delete students
- View all student records in a table

### 📊 Marks Management
- Add marks per trimester
- Display student results
- Automatic average calculation

### 💰 Fees Management
- Add fee records
- Delete fee entries
- Track payments per student

### 📅 Attendance (Optional Module)
- Record attendance (morning / evening)
- Track student presence

### 🔐 Authentication
- Simple admin login system

---

## 🛠️ Technologies Used

- Python 3
- PyQt5 (Graphical User Interface)
- SQLite (Database)

---

## 🗂️ Database Structure

The project uses 4 main tables:

- `student`
- `mark`
- `fees`
- `attendance`
